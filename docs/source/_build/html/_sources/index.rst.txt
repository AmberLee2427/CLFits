Welcome to CLFits's documentation!
===================================

**CLFits** is a command-line tool for viewing and editing the headers of FITS files. It provides a simple, scriptable interface for common header operations.

Key Features
------------

*   **View, Get, Set, and Delete** keywords in any FITS header.
*   **Target Specific HDUs** by index or by name.
*   **Search and Filter** headers for keywords matching specific patterns.
*   **Export** headers to common formats like JSON, YAML, and CSV.
*   **Cross-platform** support for Linux, macOS, and Windows.
